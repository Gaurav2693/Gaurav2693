## Hey, I'm Gaurav.

Design-engineer. I design systems and build them end-to-end.

I'm currently building **RONIN** — a taste-aware operating environment for design-engineers. Native macOS. Multi-model AI orchestration. A product that learns how you design and gets better the longer you use it.

### What I've shipped

- **RONIN** — Native macOS AI command center with multi-provider orchestration, SSE streaming, and a design intelligence layer that compounds over time. SwiftUI shell + Node.js orchestrator. *Currently in development.*
- **Design Intelligence System** — Terminal-aesthetic design workspace with 5 modules, MCP server integration, and Claude API. Live at [designintelligence.space](https://designintelligence.space).
- **3D Portfolio World** — Navigable isometric rooms built in React Three Fiber. Four explorable spaces: Design Studio, The Garage, How I Think, Prototype Lab.
- **34-Component Design System** — Production React component library with design tokens, accessibility baked in, and full Storybook documentation.
- **PlotSync** — Real estate SaaS with admin dashboard + mobile field app. Qwen vision pipeline for automated PDF-to-SVG site map conversion.
- **INERTIA** — Procedural side-scroller game with Web Audio API sound engine and physics-based interactions.

### How I work

I don't use tutorials. I build to learn. In ~8 months of AI-assisted development, I went from zero code to shipping full-stack production systems — React, SwiftUI, Node.js, TypeScript, Three.js, GSAP, XState.

I think in systems before components. I care about taste as a measurable quality, not just a vibe.
